# users-manager

## Setup

Create and fill `.env` file by example of `.env.example`


### DB creation

Install `docker` and `docker-compose` and run `docker`.

Run following command to launch MySQL server:

```bash
$ docker-compose up
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
